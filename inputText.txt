1. 제목을 나타내는 <hn> 태그 = heading 태그
: 다른 텍스트 보다 크고 진하게 표시 h1 ~ h6
ex) <h1> 레드향 </h1> || <h2> 레드향 샐러드 레시피 </h2> 등등

2.텍스트 단락을 만드는 <p>태그 , 줄을 바꾸는 <br> 태그
: 텍스트를 적절하게 단락으로 묶어서 표시하기 위해서는 <p> 이용  <br> 은 그저 줄바꿈 용도 태그

3. 분위기를 전환하는 <hr> 태그 
: 구분선

4. 인용할때 쓰는 <blockquoute> 태그 
: 다른 사람의 말이나 책의 내용을 인용할때 흔히 큰따옴표를 사용해 표시하지만 브라우저가 인용문을 인식할 수 있게 하는 태그

5. 텍스트를 강조하거나 굵게 표시하는 <strong>, <b> 태그
: 큰 차이는 없지만 경고나 주의사항처럼 중요한 내용 강조할때 <strong> 태그 ,단순히 글자만 굵게 <b> 태그

6. 기울인 텍스트를 표시하는 <em>, <i>, <cite>
: 웹 문서에서 텍스트를 이탤릭체로 표시하는 태그.

ex) <em> 이탤릭체로 강조할 텍스트 </em>
    <i> 이탤릭체로 표시할 텍스트 </i>
    <cite> 저작물 제목 </cite>


7. 추가 태그
<abbr> 줄임말을 표시 
<code> 소스 코드 표시
<del> 문서에서 삭제된 텍스트 
<ins> 문서에 추가된 텍스트
<mark> 태그를 적용한 부분에 하이라이트 표시
<small> 덧붙이는 글이나 저작권 등 부가정보를 작게 표시할때 사용
<sub> 아래 첨자를 표시
<sup> 위 첨자를 표시
