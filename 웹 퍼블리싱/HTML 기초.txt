html - Hyper Text Markup Language -> 서로 연결해주는 링크
웹 브라우저가 그림,제목,텍스트 등 구분 하기 위해서는 <태그> 가 필요함. => 이름만 봐도 의미를 알수 있어 시맨틱 태그라고 함.

태그를 쓰는 이유 :
1. 웹 브라우저가 html 코드만 보고도 어느 부분이 제목인지 본문인지 쉽게 알수 있음.
2. 문서구조가 정확히 나눠져 모바일, 스마트 기기등 다양한 화면에서 웹 문서를 표현하기 쉬움
3. 인터넷에서 웹 사이트를 검색할때 필요한 내용을 정확하게 빠르게 찾기 가능


신경쓸점 :
    1. 여는 태그와 닫는 태그를 정확히 입력할것
    2. 적당하게 들여쓰기 할것
    3. 일부 태그는 속성과 함께 사용. ex) <img src = "images/salad.jpg" alt = "레드향" width="150">
