1. 제목을 나타내는 <hn> 태그 = heading 태그
: 다른 텍스트 보다 크고 진하게 표시 h1 ~ h6
ex) <h1> 레드향 </h1> || <h2> 레드향 샐러드 레시피 </h2> 등등

2.텍스트 단락을 만드는 <p>태그 , 줄을 바꾸는 <br> 태그
: 텍스트를 적절하게 단락으로 묶어서 표시하기 위해서는 <p> 이용  <br> 은 그저 줄바꿈 용도 태그

3. 분위기를 전환하는 <hr> 태그 
: 구분선

4. 인용할때 쓰는 <blockquoute> 태그 
: 다른 사람의 말이나 책의 내용을 인용할때 흔히 큰따옴표를 사용해 표시하지만 브라우저가 인용문을 인식할 수 있게 하는 태그

5. 텍스트를 강조하거나 굵게 표시하는 <strong>, <b> 태그
: 큰 차이는 없지만 경고나 주의사항처럼 중요한 내용 강조할때 <strong> 태그 ,단순히 글자만 굵게 <b> 태그

6. 기울인 텍스트를 표시하는 <em>, <i>, <cite>
: 웹 문서에서 텍스트를 이탤릭체로 표시하는 태그.

7. 어떤 글이 있다가 제거되었음을 나타내는 <s>
: 글자에 줄로 표현 글 가운데에 줄이라고 생각하면 됨.

8. 미리 서식이 정해진 텍스트 <pre>, <code>
: HTML에 적혀있는 그대로 보여주는 태그 (둘 다 코드 사용할때 자주 사용 )

9. 순서를 적어줄때 사용하는 태그 <ol>, <li> (ordered list) +(list item)
:
1.
2.
3.
이렇게 순서가 정해질 경우 
<ol>
<li></li>
<li></li>
<li></li>
</ol> 로 표현하면 해결

type을 통해 숫자 말고도 로마 숫자, 영어도 표기 가능
<ol type = "a"> 하면 a. b. c. <ol type = "i"> i. ii. iii. iv
디자인도 추가 가능(css) <style> ol {list-style: disc; } </style> => ul태그와 동일한 행동 (불렛 표현이 disc)

10. 순서 없는 리스트 태그 <ul>, <li>   (unordered list) +(list item) - 불렛 표현을 쓰고 싶은 경우
- 가위
- 바위
- 보
이런 식의 리스트에 사용 

<ul>
<li></li>
<li></li>
<li></li>
</ul>

디자인 추가 가능 (CSS) 
ul > li{
    list-style: inline-block;
    border: 1px solid #dddddd;
    border-radius: 24px;
    padding: 8px;
} => 시각적으로는 목차처럼 나오는데 html에서는 목록이라는 의미를 해줄수 있어 많이 사용
리스트 항목을 한 줄로 쓰고 싶으면 display: inline-block을 활용할 수도 있습니다. 내가 원하는 디자인으로 보여주면서도 HTML에서는 목록이라는 의미를 분명하게 나타낼 수 있습니다.


ex) <em> 이탤릭체로 강조할 텍스트 </em>
    <i> 이탤릭체로 표시할 텍스트 </i>
    <cite> 저작물 제목 </cite>


11. 추가 태그
<abbr> 줄임말을 표시 
<code> 소스 코드 표시
<del> 문서에서 삭제된 텍스트 
<ins> 문서에 추가된 텍스트
<mark> 태그를 적용한 부분에 하이라이트 표시
<small> 덧붙이는 글이나 저작권 등 부가정보를 작게 표시할때 사용
<sub> 아래 첨자를 표시
<sup> 위 첨자를 표시
